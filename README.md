# Click => [Marvel heroes](https://bardier.github.io/marvel/)

### Учебный проект. Позволяет получать разных супергероев Marvel по API и выдает информацию по ним, также дает доступ к комиксам.

### Стек технологий
- React
- React-router
- Prop-types
- Scss

### Установка и запуск проекта
- ``` npm i ``` 
- ```npm start```
